## âœ¨ What is GaLaksi?

GaLaksi! is a mobile travel planning application designed to help users seamlessly create and manage their travel itineraries. With a focus on connectivity and personalization, the app allows users to:

- Plan trips with detailed day-to-day schedules  
- Add important travel information (flights, accommodations, notes)  
- Share travel plans with friends via QR code  
- Discover and connect with users who have similar travel interests  
- Get timely pre-trip reminders
