### Description

**Galaksi** is a travel planning app that will help turn plan your _gala_ even if it's to the mall or to the Milky Way galaxy!  
The app includes flexible travel planning features that will surely help you stay on top of your itinerary and make sure you are organized.

### Installation Guide

In order to the install this app _in development phase_, you must do the following:

1. Make sure you have **Flutter** and **Dart** installed. There are several instructions online on how to install these. [Here](https://flutter.dev/learn)
1. Clone the repository  
   `git clone https://github.com/your-org/your-repo-name.git`  
   `cd your-repo-name`
1. Install dependencies  
   `flutter pub get`
1. Setup **Firebase** database  
   `flutterfire configure --project=project-galaksi`  
   _However, you won't have access here, unless we grant it._
1. Finally, run the app  
   `flutter run`

### How to Use

1. You can either **login** or **register** with your email.
1. You will be **onboarded** for your preferences and styles (_Optional_)
1. Congrats! **You're in**. Create a brand new travel plan or **share with a friend**.
1. Creating a travel plan requires only a **title**, with an optional description. _So simple right?_
1. We're not done yet. You can create **travel activities** within the travel plans. _These are the core elements of **Galaksi**_
1. Creating travel plan requires the **title**, **dates**, **location**, and optionally: _some reminders_.
1. Once done, you can now view your existing travel plans.
1. Additionally, there are **notes**, **flight details**, and **accommodations** you can add to your travel plan to stay organized.
1. You can also try adding friends and add them to your own travel plan!
